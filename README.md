# Weather App 🌤

## Description
This project is a simple Weather Application built using HTML, CSS, and JavaScript. It fetches real-time weather data from the OpenWeather API and displays weather details for the city entered by the user.

## Features
- Search weather by city name
- Displays temperature
- Displays weather condition
- Displays humidity
- Displays wind speed
- Responsive design
- Background image with glassmorphism effect

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## API Information

API Name:
OpenWeather API

API URL:
https://api.openweathermap.org/data/2.5/weather

## How to Run

1. Download or clone the repository
2. Open the folder in VS Code
3. Install the Live Server extension
4. Right-click on index.html
5. Select "Open with Live Server"
6. Enter a city name
7. Click the Search button
